# Тестовый проект на Selenide с Allure отчетами

## Запуск автотестов

### Использование Maven

Для запуска тестов из командной строки используйте следующую команду: mvn clean test

#Генерация Allure отчёта
После успешного запуска тестов выполните команду для генерации отчёта:
allure generate target/allure-results --clean -o target/allure-report

Просмотр Allure отчёта
Для быстрого просмотра отчёта запустите локальный сервер, эта команда автоматически сгенерирует и откроет отчёт в браузере.:
allure serve target/allure-results
